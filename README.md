# ProiectTWJS

🚑 Emergency First Aid Website

An educational web project that provides clear and accessible first aid guidance for common medical emergencies. The website is designed to help users quickly find step-by-step instructions for emergency situations, improving awareness and response in critical moments.

This project was developed as part of the Web Technologies course, focusing on front-end development and user-friendly information presentation.

📌 Features

- Step-by-step first aid instructions for common emergencies

- Fast and simple navigation for quick access to information

- Educational content designed to improve emergency awareness

- Responsive and clean interface built using modern front-end technologies

- Structured medical information for easier learning and understanding

🛠️ Technologies Used

HTML5 – structure and semantic layout

CSS3 – styling and responsive design

JavaScript – interactivity and dynamic content

🎯 Purpose of the Project

The goal of this project is to create an informative and accessible platform where users can quickly learn how to respond to medical emergencies.

The website focuses on presenting clear instructions and essential first aid knowledge in a structured and easy-to-navigate format.
